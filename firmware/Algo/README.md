# Algo 模块说明

## 这个目录是干嘛的

`Algo` 用来放数据处理和识别算法，也就是“怎么从波形里判断有没有雨滴”。

## 建议放什么

- 峰峰值计算
- 能量计算
- 方差计算
- 事件识别
- 阈值判定
- 后续机器学习规则导出结果

## 以后可以新增的文件示例

- `rain_features.c`
- `rain_features.h`
- `rain_detector.c`
- `rain_detector.h`

## 设计原则

- 输入尽量是原始帧或特征数据
- 输出尽量是清晰的计算结果或判定结果
- 不直接耦合底层硬件寄存器

## 一句话理解

`Algo` 负责“把数据变成结论”。
