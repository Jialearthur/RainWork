# 洞洞板新手教学：模电区和数电区怎么分开

这份说明是给第一次做洞洞板原型的人看的。目标不是一次焊出最终版本，而是先把布局思路搞清楚，避免一开始把线走乱。

注意：如果和项目给定布局图不一致，实际焊接优先按项目给定布局图执行。项目图对应的详细解释见 `hardware/schematics/provided_layout_beginner_reading.md`。

配套草图：

- `hardware/schematics/perfboard_analog_digital_layout.svg`

## 1. 先理解什么是模电，什么是数电

### 模电部分

模电就是处理连续电压变化的部分。

在这个项目里，模电部分包括：

- 红外 LED
- 光电二极管
- LM358 运放
- 电阻、电容、RC 滤波
- 从光电二极管到 LM358 输入端的微弱信号线

它的特点是：

- 信号比较弱
- 容易被干扰
- 走线越短越好
- 旁边尽量不要放高速数字线

### 数电部分

数电就是处理数字信号和单片机控制的部分。

在这个项目里，数电部分包括：

- STM32F407
- USB-TTL 串口模块
- ST-Link 下载接口
- 串口线 `PA9/PA10`
- ADC 引脚 `PA0`
- 电源入口

它的特点是：

- 信号跳变快
- 电流变化更明显
- 容易把干扰传到模电部分

所以我们要把模电和数电物理分开。

## 2. 洞洞板上怎么分区

通用原则是假设洞洞板横着放：

```text
左边                         中间                  右边
模电区                       隔离带                数电区

红外LED / 光电二极管          尽量不走线            STM32F407
LM358 / RC滤波                至少1cm              USB-TTL
微弱模拟信号                                       电源入口
```

通用推荐布局：

- 左边 `40%` 放模电
- 中间 `10%` 留空
- 右边 `50%` 放数电

中间隔离带不是必须完全空着，但新手阶段建议尽量不走线。

## 3. 元件摆放顺序

先不要急着焊，先把元件插在洞洞板上比划位置。

推荐顺序：

1. 如果 STM32F407 直接上洞洞板，就放在右侧。
2. 如果按项目给定布局图，STM32F407 不上洞洞板，只通过底部 `4P 排针` 外接。
3. LM358 放在左侧偏中间。
4. 光电二极管放在 LM358 输入附近。
5. 红外 LED 驱动放在右侧，远离接收放大区。
6. 电阻电容围绕 LM358 放，不要离太远。

原则：

- 光电二极管到 LM358 输入端要短。
- LM358 输出到 `PA0` 可以稍微长一点。
- USB-TTL 串口线不要穿过光电二极管输入区。

## 4. 第一版只做一路

新手不要一上来做 5 路。

第一版只做：

```text
1 个红外 LED
1 个光电二极管
1 路 LM358
1 根输出线到 PA0
```

这样有三个好处：

- 接线少，容易排错。
- ADC 数值变化更容易看懂。
- 一路跑通后，再复制到多路。

## 5. 最小接线关系

### 红外发射端

```text
3.3V -> 220R 电阻 -> 红外 LED 阳极
红外 LED 阴极 -> GND
```

作用：

- 红外 LED 持续发光。
- `220R` 用来限流，防止 LED 直接烧坏。

### 光电接收端

先做一个信号点，叫 `SENSE`：

```text
3.3V -> 10k 电阻 -> SENSE
SENSE -> 光电二极管 -> GND
```

作用：

- 光照变化会让 `SENSE` 电压变化。
- 后面 LM358 就读取这个 `SENSE`。

如果数值变化方向反了，可以把光电二极管方向反过来试。

### LM358 电压跟随器

先用 LM358 的第一路运放。

常见 DIP-8 引脚：

```text
1脚：输出1
2脚：反相输入1
3脚：同相输入1
4脚：GND
8脚：VCC
```

最小接法：

```text
LM358 8脚 -> 3.3V
LM358 4脚 -> GND
LM358 1脚 <-> LM358 2脚
LM358 3脚 -> SENSE
LM358 1脚 -> STM32 PA0
```

这里 LM358 不是先放大，而是先做缓冲。

好处：

- 不容易自激。
- 不容易输出超过 ADC 范围。
- 方便先确认信号链路能跑通。

## 6. 地线怎么走

地线是新手最容易忽略的地方。

推荐做法：

```text
洞洞板边缘焊一圈 GND 铜丝
模电区地线接左边地线
数电区地线接右边地线
最后只在一个点连起来
```

这叫单点共地。

至少要保证这些地方共地：

- STM32 GND
- USB-TTL GND
- LM358 4脚
- 红外 LED GND
- 光电二极管 GND

如果不共地，ADC 会读到乱跳的数值。

## 7. 焊之前先画这几个点

在草稿纸上先标：

```text
左边：红外 LED、光电二极管、LM358、RC
右边：STM32、USB-TTL、电源
中间：隔离带
底边：GND 总线
```

然后画 4 条关键线：

```text
红外 LED 供电线
光电二极管到 LM358 输入线
LM358 输出到 PA0 线
GND 总线
```

如果这 4 条线画清楚，第一版板子就不会乱。

## 8. 上电前检查清单

不要直接上电，先检查：

- `3.3V` 有没有短到 `GND`
- LM358 的 `8脚` 是否接 `3.3V`
- LM358 的 `4脚` 是否接 `GND`
- LM358 的 `1脚` 和 `2脚` 是否短接
- LM358 的 `1脚` 是否接到 `PA0`
- 光电二极管是否接在 `SENSE` 和 `GND` 之间
- STM32、LM358、USB-TTL 是否共地
- `PA0` 输入有没有超过 `3.3V`

## 9. 上电后怎么判断对不对

当前程序会打印：

```text
ADC(PA0)=xxxx
```

测试顺序：

1. 不接 LM358，先测 `PA0 -> GND`，应该接近 `0`。
2. 不接 LM358，测 `PA0 -> 3.3V`，应该接近 `4095`。
3. 接上 LM358 输出，观察静止时数值。
4. 用手遮挡红外光路，看数值是否变化。
5. 用白纸靠近反射，看数值是否变化。

只要数值会跟动作变化，就说明第一版链路通了。

## 10. 常见错误

### 串口正常，但 ADC 一直是 0

可能原因：

- `PA0` 被接到 GND。
- LM358 没供电。
- LM358 输出脚接错。
- 光电二极管方向导致信号点被拉死。

### ADC 一直接近 4095

可能原因：

- `PA0` 被接到 3.3V。
- LM358 输出顶到高电平。
- 光电二极管方向或电阻连接有问题。

### ADC 乱跳很厉害

可能原因：

- `PA0` 悬空。
- 没有共地。
- 模电输入线太长。
- USB-TTL 或数字线穿过模电输入区。

### 遮挡没有变化

可能原因：

- 红外 LED 没亮。
- 光电二极管方向反了。
- 发射管和接收管角度不对。
- 黑盒遮光结构还没做好，环境光太强。

## 11. 新手阶段的目标

第一天不要追求“能测雨量”。

第一天只追求：

```text
遮挡或反射变化时，ADC(PA0) 数值有明显变化
```

这一步成功后，再进入：

```text
滴水测试 -> 保存数据 -> Python 画波形 -> 算特征
```
