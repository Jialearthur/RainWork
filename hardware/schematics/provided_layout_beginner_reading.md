# 项目给定洞洞板布局新手解读

这份说明按你截图里的布局来讲。实际焊接时优先按这张项目图执行。

## 先看整体

这张图不是把 `STM32F407` 直接放到洞洞板上，而是把洞洞板当成一个传感器前端小板。

洞洞板上主要分成两边：

- 左边：接收 + 放大区，属于敏感模电区。
- 右边：PWM + LED 驱动区，属于噪声源区。
- 中间：虚线隔离带，尽量不走线。
- 底部：`4P 排针` 接到 `F407`。

所以它真正的结构是：

```text
左边接收放大  ->  底部 4P 排针  ->  F407
右边 LED 驱动 <-  底部 4P 排针  <-  F407 PWM
```

## 为什么左边放接收和 LM358

左边蓝色区域是最敏感的地方。

包含：

- `D2`：光电二极管 `PD204`
- `U1`：`LM358`
- `R3/R4/R5`
- `C1`
- 反馈和输出元件群：`R6/R7/R10/R11/R12/R9`、`C2/C3`
- `C4/C5`：LM358 退耦电容

这部分处理的是很弱的接收信号，所以要远离右边的 LED 开关驱动。

新手重点：

- 光电二极管到 LM358 输入的线要短。
- LM358 旁边的 `100nF` 电容要靠近芯片电源脚。
- 左边不要穿过 PWM 线、串口线、下载线。

## 为什么右边放 PWM + LED 驱动

右边红色区域是噪声比较大的地方。

包含：

- `Q1`：`S8050`
- `D1`：红外 LED `IR204C`
- `R1/R2`
- `C6`：电源入口大电容

这部分由 `F407` 的 `PWM` 控制红外 LED 开关，电流变化比较明显，会产生干扰。

所以它被放到右边，和左边接收放大区拉开距离。

## 底部 4P 排针怎么理解

图里底部写的是：

```text
4P 排针 -> 杜邦线 -> F407
```

四个信号是：

```text
3V3
GND
PWM
SIG
```

含义：

- `3V3`：F407 提供给洞洞板的 `3.3V`
- `GND`：公共地
- `PWM`：F407 输出给右边 LED 驱动区
- `SIG`：左边 LM358 输出回 F407 的 ADC 输入

建议对应：

```text
3V3 -> F407 3.3V
GND -> F407 GND
PWM -> F407 的 PWM 输出脚
SIG -> F407 PA0 ADC 输入
```

当前我们程序正在测 `PA0`，所以 `SIG` 先接 `PA0`。

## 中间虚线是什么

中间虚线是模电和噪声源的分界线。

新手可以这样理解：

```text
左边尽量安静
右边允许有 PWM 和 LED 电流
中间尽量别乱穿线
```

跨过中线的线越少越好。

必须跨过去的主要是：

- 电源
- 地
- 从 LM358 输出到排针的 `SIG`
- 从排针到 LED 驱动的 `PWM`

## 橙色单点连接是什么

图底部中间有一个橙色点，写着：

```text
模拟地 + 数字地 单点连接
用 0Ω 电阻或一段铜丝
```

这表示左边的模拟地和右边的数字地不要到处乱连，最好只在这个位置连一次。

这样做是为了减少右边 PWM 电流对左边接收放大的干扰。

## 顶部 3.3V 总线

图顶部红线是：

```text
+3.3V 总线，铜丝
```

也就是沿板子上方焊一根供电铜丝。

注意：

- 它给左边 LM358 和右边 LED 驱动供电。
- 每个芯片附近都要有退耦电容。
- 供电线可以长一些，但输入信号线不能长。

## 测试点 TP1 / TP2 / TP3

图中间靠左有三个测试点：

- `TP1`：接收原始
- `TP2`：一级输出
- `TP3`：二级输出

新手阶段它们很有用。

如果后面 ADC 没变化，可以按顺序测：

```text
先看 TP1 有没有变化
再看 TP2 有没有变化
最后看 TP3 / SIG 有没有变化
```

这样能知道问题卡在光电接收、一级放大，还是二级放大。

## 实际焊接顺序

建议按这个顺序来：

1. 先焊顶部 `3.3V` 铜丝和底部/边缘 `GND`。
2. 焊底部 `4P 排针`：`3V3/GND/PWM/SIG`。
3. 焊左边 LM358 和退耦电容 `C4/C5`。
4. 焊左边光电二极管 `D2` 和输入电阻电容。
5. 焊左边反馈/输出元件群。
6. 焊右边 `S8050`、红外 LED、限流/基极电阻。
7. 最后做模拟地和数字地单点连接。

## 现在你应该怎么接到 F407

先只做最小测试：

```text
洞洞板 3V3 -> F407 3.3V
洞洞板 GND -> F407 GND
洞洞板 SIG -> F407 PA0
洞洞板 PWM 先不接，或者后面再接
```

因为我们当前程序只是测试 `PA0 ADC`，先确认左边接收放大输出是否能被 `F407` 读到。

等 `SIG -> PA0` 能看到变化后，再接 `PWM` 控制红外 LED。

## 这张图和之前通用图的关系

之前的图讲的是原则：

```text
模电和数电要分开
噪声源远离敏感输入
地线单点连接
```

你截图这张是具体执行版本：

```text

```

实际做板子时，以截图这张具体布局为准。
